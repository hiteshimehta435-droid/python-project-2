# Python Project 2

## Project Title
Menu Driven Pattern Generator and Number Analyzer

## Objective
This project demonstrates the use of:
- Loops
- Control Statements
- Break Statement
- Pass Statement
- Menu-Driven Programming

## Features
1. Generate a Right-Angled Triangle Pattern.
2. Analyze numbers in a given range.
3. Exit the program.

## Control Statements Used
### break
Used to stop the program when an invalid row count is entered or when the user chooses Exit.

### pass
Used as a placeholder statement in the number analysis section.

## How to Run
1. Save the code as `project2.py`
2. Open Terminal or Command Prompt.
3. Run:

```bash
python project2.py
